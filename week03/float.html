Floating an Image

This paragraph wraps around the image because the image is floated to the right. The float property allows text to flow around visual elements, such as photos or diagrams, while keeping the layout compact.

Floats are often used for simple layouts or to place images next to text. In modern layouts we usually prefer Flexbox or CSS Grid, but understanding floats is still important when reading or maintaining older code.

Float & Clearfix

Floating an Image Right

Este párrafo envuelve una imagen flotada a la derecha. float se usaba para maquetación antigua; hoy se prefiere Flexbox y Grid, pero sigue útil para que el texto rodee imágenes como en este ejemplo.

El helper .clearfix usa un pseudo-elemento para limpiar el float y que el contenedor abarque la imagen flotada.
